# Sales Prediction Using Machine Learning

## Project Overview
This project predicts sales based on advertising budgets spent on TV, Radio, and Newspaper marketing channels. A Linear Regression model is used to analyze the relationship between advertising expenditure and sales.

## Dataset
The dataset contains 200 records with the following features:
TV: Advertising budget spent on TV
Radio: Advertising budget spent on Radio
Newspaper: Advertising budget spent on Newspaper
Sales: Total sales generated

## Technologies Used
Python
Pandas
NumPy
Matplotlib
Scikit-learn
Linear Regression

## Project Workflow

### 1. Data Collection
Loaded the Advertising.csv dataset.

### 2. Data Preprocessing
Removed unnecessary columns.
Checked for missing values.
Verified data types.

### 3. Data Visualization
Created scatter plots to analyze:
TV vs Sales
Radio vs Sales
Newspaper vs Sales

### 4. Model Training
Used Linear Regression to train the model on advertising data.

### 5. Model Evaluation
Evaluated the model using R² Score.
R² Score Achieved:0.8994

### 6. Sales Prediction
The model predicts future sales based on user-provided advertising budgets.

## Example Input
TV Advertising Budget: 200
Radio Advertising Budget: 30
Newspaper Advertising Budget: 40

## Example Output
Predicted Sales: 17.71

## Results
TV advertising shows a strong positive impact on sales.
Radio advertising also contributes significantly to sales.
Newspaper advertising has a comparatively lower impact.
The model achieved approximately 90% prediction accuracy.

## Conclusion
This project demonstrates how Machine Learning can be used to forecast sales based on advertising expenditures. Businesses can use such predictions to make better marketing and budgeting decisions.
